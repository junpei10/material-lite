Directives

selector：[mlRipple]
exportAs：mlRipple

@Input

Selector Name Description
mlRipple isEnabled Rippleの有効・無効を設定する。
mlRippleOverdrive overdrive booleanを代入すると、Overdriveを有効にするかどうかを設定する。
Objectを代入すると、クリック時、コンテナのサイズが設定したサイズより小さい場合は通常の"Ripple"を表示し、同じか大きい場合は"Overdrive"を表示するようになる。
mlRippleTheme theme Rippleの背景色を設定済みのTheme用に設定する。
mlRippleColor color Rippleの背景色を指定した色に設定する。
優先度はThemeのほうが上になる。
mlRippleOpacity opacity Rippleの透明度を設定する。
mlRippleRadius radius Rippleの半径を設定する。指定されていない場合、クリックした箇所からの最も遠い角までの距離になる。
mlRippleAnimationDuration animationDuration 波紋が広がる速度(enter)、波紋が消える速度(leave)を設定する。
mlRippleTrigger trigger 指定したターゲットにPointerdownのイベントリスナーを追加し、クリックすると出力先からRippleが出力されるようになる。
mlRippleTriggerIsOutside triggerIsOutside mlRippleTriggerを使用したとき、TriggerがOutlet(出力先・Rippleが表示される部分)より外側にある場合遅延が生じるが、その遅延を無効にする。
デメリットとして、Rippleを出力する毎に計算をする必要があるため処理が遅くなる。mlRippleTriggerを用いてる場合のみ有効。
mlRippleCentered centered Rippleが出現する場所が中心になる。
mlRippleFixedSize fixedSize コンテナのサイズが変わらない場合、指定することで処理を軽量化できる。
