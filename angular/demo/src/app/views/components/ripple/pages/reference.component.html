@Input

Selector Type Description
mlRipple boolean Rippleの有効・無効を設定する。
mlRippleOverdrive boolean Overdriveを有効にするかどうかを設定うする。
mlRippleTypeBreakpoint object クリック時、コンテナのサイズが設定したサイズより同じか小さい場合は通常の"Ripple"を表示し、大きい場合は"Overdrive"を表示するようになる。
mlRippleTheme string Rippleの背景色を設定済みのTheme用に設定する。
mlRippleColor string Rippleの背景色を指定した色に設定する。
優先度はThemeのほうが上になる。
mlRippleOpacity number Rippleの透明度を設定する。
mlRippleRadius number Rippleの半径を設定する。指定されていない場合、クリックした箇所からの最も遠い角までの距離になる。
mlRippleAnimationDuration number | object 波紋が広がる速度(enter)、波紋が消える速度(leave)を設定する。
mlRippleTarget ListenTarget 指定したターゲットにPointerdownのイベントリスナーを追加し、クリックすると出力先からRippleが出力されるようになる。
mlRippleTargetIsOutside boolean mlRippleTargetを使用したとき、TargetがOutlet(出力先・Rippleが表示される部分)より外側にある場合遅延が生じるが、その遅延を無効にする。
デメリットとして、Rippleを出力する毎に計算をする必要があるため処理が遅くなる。mlRippleTargetを用いてる場合のみ有効。
mlRippleCentered boolean Rippleが出現する場所が中心になる。
mlRippleFixedSize boolean コンテナのサイズが変わらない場合、指定することで処理を軽量化できる。
