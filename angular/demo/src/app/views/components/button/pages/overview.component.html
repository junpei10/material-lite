Types

ボタンのタイプを変えたいときは[variant]を変更します。

<div>
  <button mlButton>Default</button>
  <button mlButton variant="basic">Basic</button>
  <button mlButton variant="raised">Raised</button>
  <button mlButton variant="stroked">Stroked</button>
  <button mlButton variant="flat">Flat</button>
  <button mlButton variant="icon"><span mlIcon>home</span></button>
  <button mlButton variant="fab"><span mlIcon>home</span></button>
</div>

Theming

[theme]を変更することで、色を変えることができます

<div>
  <button mlButton variant='basic' theme='primary'>Basic</button>
  <button mlButton variant='raised' theme='accent'>Raised</button>
  <button mlButton variant='stroked' theme='warn'>Stroked</button>
  <button mlButton variant='flat' theme='primary'>Flat</button>
  <button mlButton variant='icon' theme='accent'><span mlIcon>home</span></button>
  <button mlButton variant='fab' theme='warn'><span mlIcon>home</span></button>
</div>

Link button

リンクボタンは２パターン存在し、"Hover時の背景色"と"Rippleの色"が異なっています。

a-tag Button 
<div>
  <a mlButton>a-tag Button</a>
  <button mlButton [inlineLink]="true">
    <a>Inline Link Button</a>
  </button>
</div>

Disable button

入力プロパティの数をできるだけ少なくしたいという考えから、[mlButton]に値を代入することで変更できるようにしています。

<div>
  <button [mlButton]="false" variant="basic">Disabled</button>
  <button [mlButton]="!true" variant="raised">Disabled</button>
  <button [mlButton]="undefined" variant="stroked">Disabled</button>
  <button [mlButton]="null" variant="icon"><span mlIcon>home</span></button>
  <button [mlButton]="0" variant="fab"><span mlIcon>home</span></button>
</div>

Hover action

Hover時の背景色の設定を変えることができます。
例えば、variant="raised"を入力したボタンの場合、Hoverしても背景色は変わりませんが、 hoverAction="enable"を入力すると、Hoverしたときに背景色が変わるようになります。 

<div>
  <button mlButton variant="basic" hoverAction="disable">Disable Hover Action</button>
  <button mlButton variant="raised" hoverAction="enable">Enable Hover Action</button>
</div>

Ripple config

３つのRippleの設定([rippleDisabled], [rippleOverdrive], [rippleTypeBreakpoint])を変更可能です。
詳しくはRippleを参照してください！ 

<div>
  <button mlButton [disableRipple]=true>Disable Ripple</button>
  <button mlButton [immediateRipple]=true>Immediate Ripple</button>
</div>
