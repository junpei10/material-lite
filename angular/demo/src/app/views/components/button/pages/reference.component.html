@Input

Selector Type Description
mlButton boolean Disabled用のスタイルにするかどうかを設定する。
theme string ボタンの配色を設定済みのTheme用に設定する。
variant string ボタンのスタイルを設定する。
hoverAction string Hover時の背景色をスタイルを設定する。
wrapAnchor boolean ボタンの内部に<a>を設置する場合のスタイルを設定する。
rippleDisabled boolean Rippleを無効化する。
rippleOverdrive boolean RippleをのタイプをOverdriveにする。
rippleTypeBreakpoint object クリック時、ボタンのサイズが設定したサイズより同じか小さい場合は通常の"Ripple"を表示し、大きい場合は"Overdrive"を表示するようになる。

  wrapAnchor?: boolean;
  rippleDisabled?: boolean;
  rippleOverdrive?: boolean;
  rippleTypeBreakpoint?: {{ "{" }}
    width?: number,
    height?: number
