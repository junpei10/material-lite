Directives

selector：[mlButton]
exportAs：mlButton

@Input

Selector Name Description
mlButton isEnabled Disabled用のスタイルにするかどうかを設定する。
theme theme ボタンの配色を設定済みのTheme用に設定する。
variant variant ボタンのスタイルを設定する。
hoverAction hoverAction Hover時の背景色をスタイルを設定する。
wrapAnchor wrapAnchor ボタンの内部に<a>を設置する場合のスタイルを設定する。
rippleDisabled rippleDisabled Rippleを無効化する。
rippleOverdrive rippleOverdrive RippleをのタイプをOverdriveにする。
クリック時、ボタンのサイズが設定したサイズより小さい場合は通常の"Ripple"を表示し、同じか大きい場合は"Overdrive"を表示するようになる。 
@Input('mlButton') isEnabled?: boolean | '';
@Input() theme?: string;
@Input() variant?: 'basic' | 'raised' | 'stroked' | 'flat' | 'fab' | 'icon';
@Input() hoverAction?: 'auto' | 'enable' | 'disable';
@Input() wrapAnchor?: boolean;
@Input() rippleDisabled?: boolean;
@Input() rippleOverdrive?: boolean | {{ "{" }}
  width?: number,
  height?: number
{{ "}" }};
