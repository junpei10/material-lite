Directives

@Attribute

Selector Type Description
key string 設定すると、"PortalOutlet"Serviceを用いてアクセスすることが可能となり、どこからでも出力できる。

@Input

Selector Type Description
mlPortalOutlet MlPortalAttachContent
mlPortalOutletConfig MlPortalAttachConfig [mlPortalOutlet]に値が代入され、Portalから出力した際に参照される設定。Portalから値が出力される際にしか参照されないため、Portalが出力されたあとに変更しても再出力されない限り設定は変わらない。

@Output

Selector Type Description
mlPortalAttachedRef MlPortalAttachedRef 出力されるタイミングは、[mlPortalOutlet]に値が代入され、Portalから出力したとき。
