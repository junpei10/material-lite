Directives

selector：[mlPopupOutlet]
exportAs：mlPopupOutlet

@Attribute

Selector Description
key 設定すると、"PopupOutlet"Serviceを用いてアクセスすることが可能となり、どこからでも出力できる。 
@Attribute('key') key: string;

@Input

Selector Description
mlPopupOutlet
mlPopupOutletConfig [mlPopupOutlet]に値が代入され、Popupから出力した際に参照される設定。Popupから値が出力される際にしか参照されないため、Popupが出力されたあとに変更しても再出力されない限り設定は変わらない。 
@Input('mlPopupOutlet') content: MlPortalContent;
@Input('mlPopupOutletConfig') config: MlPopupConfig;

@Output

Selector Description
mlPopupAttachedRef 出力されるタイミングは、[mlPopupOutlet]に値が代入され、Popupから出力したとき。

Services

mlPopupOutlet

Property Description
attach 第２引数に代入した値に対応した出力先(outlet)に、第１引数に代入したコンテンツを出力する。
attachContents 複数の値を同時に出力する。戻り値は配列で出力される。
detachAll 第１引数に代入した値に対応した出力先(outlet)に出力済みのコンテンツをすべて削除する。

outletKey outletのkeyの名前。

setBackdrop backdropを表示するか非表示するかを切り替える。
