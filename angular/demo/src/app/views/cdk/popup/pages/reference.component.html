Directives

@Attribute

Selector Type Description
key string 設定すると、"PopupOutlet"Serviceを用いてアクセスすることが可能となり、どこからでも出力できる。

@Input

Selector Type Description
mlPopupOutlet MlPopupAttachContent
mlPopupOutletConfig MlPopupAttachConfig [mlPopupOutlet]に値が代入され、Popupから出力した際に参照される設定。Popupから値が出力される際にしか参照されないため、Popupが出力されたあとに変更しても再出力されない限り設定は変わらない。

@Output

Selector Type Description
mlPopupAttachedRef MlPopupAttachedRef 出力されるタイミングは、[mlPopupOutlet]に値が代入され、Popupから出力したとき。
