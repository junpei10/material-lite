Overview

テーマをセットする方法は、 main.ts 等の初期のタイミングで読み込まれるファイルを使います。
任意のファイルで MlTheming.init([...]) 関数を呼び出します。

以下は、用意されているテンプレートを用いたテーマの設定の一例です。

import {{ "{" }} MlTheming, ML_LIGHT_THEME, ML_DARK_THEME, ML_INDIGO_PINK_PALETTE, ML_DEEPPURPLE_AMBER_PALETTE {{ "}" }} from '@material-lite/angular/core';

MlTheming.init([
  {{ "{" }} theme: ML_LIGHT_THEME, palette: ML_INDIGO_PINK_PALETTE {{ "}" }},
  {{ "{" }} theme: ML_DARK_THEME, palette: ML_DEEPPURPLE_AMBER_PALETTE, wrapperClass: 'dark-theme' {{ "}" }},
]);

MlTheming.init type

interface InitArg {{ "{" }}
  theme: MlTheme;
  palette: MlPalette;
  wrapperClass?: string;
{{ "}" }}

abstract init(themes: InitArg[]);

Theme color

Example

import {{ "{" }} MlTheming, MlTheme, ML_LIGHT_THEME, ML_INDIGO_PINK_PALETTE {{ "}" }} from '@material-lite/angular/core';

const myTheme: MlTheme = {{ "{" }}
  ...ML_LIGHT_THEME,
  primaryContainer: 'white',
  divider: '#BDBDBD',
  text: 'rgba(0,0,0,.87)'
{{ "}" }}

MlTheming.init([
  {{ "{" }} theme: myTheme, palette: ML_INDIGO_PINK_PALETTE {{ "}" }}
]);

Type

interface MlTheme {{ "{" }}
  base: string;
  oppositeBase: string;
  background: string;
  primaryContainer: string;
  secondaryContainer: string;
  tertiaryContainer: string;
  disabledContainer: string;
  divider: string;
  elevation: string;
  scrollbar: string;
  text: string;
  secondaryText: string;
  hintText: string;
  disabledText: string;
{{ "}" }}

Palette color

Javascriptを用いてテーマを設定するため、テーマのパレット(primary, accent, warn など)を増やす事ができます。

Example

import {{ "{" }} MlTheming, MlPalette, ML_LIGHT_THEME, ML_INDIGO_PINK_PALETTE {{ "}" }} from '@material-lite/angular/core';

const myPalette: MlPalette = {{ "{" }}
  ...ML_INDIGO_PINK_PALETTE,
  secondary: {{ "{" }}
    color: '#8BC34A';
    contrast: 'rgba(0,0,0,.87)';
  {{ "}" }}
{{ "}" }}

MlTheming.init([
  {{ "{" }} theme: ML_LIGHT_THEME, palette: myPalette {{ "}" }}
]);

Type

interface MlPalette {{ "{" }}
  [palette: string] {{ "{" }}
    color: string;
    contrast: string;
  {{ "}" }}
{{ "}" }}

Palette color class

MlTheming.init関数を呼び出すと、paletteの色に対応したclassが追加されます。

primary(color: '#673AB7', contrast: 'black') が設定されているときの例： 

.ml-primary {{ "{" }}
  background: #673AB7;
  color: black;
{{ "}" }}

.ml-primary-bg {{ "{" }}
  background: #673AB7;
{{ "}" }}

.ml-primary-text {{ "{" }}
  color: #673AB7;
{{ "}" }}

.ml-primary-contrast {{ "{" }}
  color: black;
{{ "}" }}

Wrapper class

<body>などのタブに、指定したクラスを付与することでスタイルが反映されます。重複させることはできません。

import {{ "{" }} MlTheming, ML_DARK_THEME, ML_DEEPPURPLE_AMBER_PALETTE {{ "}" }} from '@material-lite/angular/core';

MlTheming.init([
  {{ "{" }} theme: ML_DARK_THEME, palette: ML_DEEPPURPLE_AMBER_PALETTE, wrapperClass: 'dark-theme' {{ "}" }}
]);
