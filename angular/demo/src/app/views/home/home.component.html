Material Lite

for

Angular

"Angular Material"の"Change detection"の発生を抑えたComponents

v0.1.1

インストール

    npm install @material-lite/angular @material-lite/angular-cdk

概要

デザインや機能はできるだけ"Angular Material"と同じパフォーマンスを提供できるよう作成しています。 "Material lite"の一番の特徴は"Change detection"を抑え、"Theme"のカスタマイズが豊富であることです。 v0 では、２種のComponent(Button・Ripple)と２種のCDK(Portal・Popup)を提供しています。 また、未実装ですが、Breakpointを変更するだけで自動的にレスポンシブデザインをスタイリングしてくれる機能を実装中です。
