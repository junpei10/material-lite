Directives

selector：[mlPortalOutlet]
exportAs：mlPortalOutlet

@Input

Selector Variable Description
mlPortalOutlet content コンテンツが代入されたときに出力する。１つしか出力できない。
mlPortalOutletAttachConfig attachConfig [mlPortalOutlet]にコンテンツが代入され、Portalから出力した際に参照される設定。Portalから値が出力される際にしか参照されないため、Portalが出力されたあとに変更しても再出力されない限り設定は変わらない。
mlPortalOutletKey key PortalOutlet serviceを用いることで、どこからでも出力できるようになるために必要な、ユニークな値を代入する。
mlPortalOutletDestroyingDuration - none - portalに対してBrowserAnimationsModuleを用いたアニメーションを付与している際、portalのtransition classesアニメーションではなく、そちらを優先して行うための設定

@Output

Selector Variable $event Description
mlPortalOutletAttached attachedEmitter MlPortalAttachedRef 出力されるタイミングは、[mlPortalOutlet]に値が代入され、Portalから出力したとき。

Services

MlPortalOutlet

Property Description
attach 第２引数に代入した値に対応した出力先(outlet)に、第１引数に代入したコンテンツを出力する。
detachAll 第１引数に代入した値に対応した出力先(outlet)に出力済みのコンテンツをすべて削除する。
detachAllOfPortalData PortalData内に記録されているPortalをすべて削除する。
hasPortalData 第１引数に代入した値のデータが存在するかどうか
setPortalData 新しいOutletのデータを保存する。

lifecycle.beforeDetached デタッチされた瞬間を伝えるObservableを出力する関数。completeは呼び出されない。
