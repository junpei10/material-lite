Directives

selector：[mlPortalOutlet]
exportAs：mlPortalOutlet

@Attribute

Selector Name Description
key key 設定すると、"PortalOutlet"Serviceを用いてアクセスすることが可能となり、どこからでも出力できる。 
@Attribute('key') key: string;

@Input

Selector Name Description
mlPortalOutlet content コンテンツが代入されたときに出力する。１つしか出力できない。
mlPortalOutletConfig config [mlPortalOutlet]にコンテンツが代入され、Portalから出力した際に参照される設定。Portalから値が出力される際にしか参照されないため、Portalが出力されたあとに変更しても再出力されない限り設定は変わらない。 
@Input('mlPortalOutlet') content: MlPortalContent;
@Input('mlPortalOutletConfig') config: MlPortalConfig;

@Output

Selector Name Description
mlPortalOutletAttached attachedEmitter 出力されるタイミングは、[mlPortalOutlet]に値が代入され、Portalから出力したとき。 
@Input('mlPortalOutletAttached') attachedEmitter: MLPortalAttachedRef;
