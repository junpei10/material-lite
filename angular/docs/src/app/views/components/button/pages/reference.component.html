Directives

selector：[mlButton]
exportAs：mlButton

@Input

Selector Variable Description
mlButton isEnabled Disabled用のスタイルにするかどうかを設定する。
theme theme ボタンの配色を設定済みのTheme用に設定する。
variant variant ボタンのスタイルを設定する。
hoverAction hoverAction Hover時の背景色をスタイルを設定する。
wrapAnchor hasWrappedAnchor ボタンの内部に<a>を設置する場合のスタイルを設定する。
disableRipple rippleIsDisabled Rippleを無効化する。
rippleConfig rippleConfig RippleをのタイプをOverdriveにする。
クリック時、ボタンのサイズが設定したサイズより小さい場合は通常の"Ripple"を表示し、同じか大きい場合は"Overdrive"を表示するようになる。
