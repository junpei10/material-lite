Type

ボタンのタイプを変えたいときは[variant]を変更します。

Theming

[theme]を変更することで、色を変えることができます

Anchor button

リンクボタンは２パターン存在し、Hover時の背景色とRippleの色が異なっています。

Anchor

Disabled button

[mlButton]に値を代入することで変更できます。

Hover action

Hover時の背景色の設定を変えることができます。
例えば、variant="raised"を入力したボタンの場合、Hoverしても背景色は変わりませんが、 hoverAction="enable"を入力すると、Hoverしたときに背景色が変わるようになります。

Ripple config

[disableRipple], [rippleConfig]の２つのRippleの設定を変更可能です。
詳しくはRippleを参照してください。
