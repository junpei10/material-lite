Directives

selector：[mlRipple]
exportAs：mlRipple

@Input

Selector Variable Description
mlRipple isEnabled Rippleの有効・無効を設定する。
mlRippleOverdrive overdrive booleanを代入すると、Overdriveを有効にするかどうかを設定する。
Objectを代入すると、クリック時、コンテナのサイズが設定したサイズより小さい場合は通常の"Ripple"を表示し、同じか大きい場合は"Overdrive"を表示するようになる。
mlRippleTheme theme Rippleの背景色を設定済みのTheme用に設定する。
mlRippleColor color Rippleの背景色を指定した色に設定する。
優先度はThemeのほうが上になる。
mlRippleOpacity opacity Rippleの透明度を設定する。
mlRippleRadius radius Rippleの半径を設定する。指定されていない場合、クリックした箇所からの最も遠い角までの距離になる。
mlRippleAnimation animation 波紋が広がる速度(enter)と、波紋が消える速度(leave)を設定する。
mlRippleEntrance entrance Rippleが入口を設定。
mlRippleTrigger trigger 指定したターゲットにPointerdownのイベントリスナーを追加し、クリックすると出力先からRippleが出力されるようになる。初期値は"host"。
