Type

Rippleには、通常の Ripple と OverdriveRipple の２種類が存在します。
主なOverdriveRippleの用途としては、コンテナのサイズが大きすぎる場合のアニメーションが窮屈になる、速くなりすぎるのを防ぐためです。

Default
Overdrive
<div>
  <div mlRipple>Default</div>
  <div mlRipple [mlRippleOverdrive]=true>Overdrive</div>
</div>

Type Breakpoint

コンテナのサイズからRippleのTypeを変更することができます。
クリック時、コンテナのサイズが設定したサイズより小さい場合は通常の"Ripple"を表示し、同じか大きい場合は"Overdrive"を表示するようになる。

Width： {{ typeBpRippleWidth }} px ・ Height: {{ typeBpRippleHeight }}px
Ripple type： {{ typeBpRippleHeight === 500 || typeBpRippleWidth === 500 ? "Overdrive" : "Ripple" }} 
<div mlRipple [mlRippleOverdrive]="{{ "{" }} width: 500; height: 500; {{ "}" }}"></div>

Trigger

Rippleを呼び出せるElementを追加できます。
[mlRippleTriggerIsOutside]=true に設定することで、Triggerが外側にあるときに発生する遅延を無くす事ができます。

Trigger
Trigger2
<div>
  <div mlRipple [mlRippleTrigger]='listenerTarget'></div>
  <div #listenerTarget>Trigger</div>
</div>

<div>
  <div mlRipple [mlRippleTrigger]='listenerTarget2' [mlRippleTriggerIsOutside]=true></div>
  <div #listenerTarget2>Trigger2</div>
</div>
