Type

Rippleには、通常の Ripple と OverdriveRipple の２種類が存在します。
主なOverdriveRippleの用途としては、コンテナのサイズが大きすぎる場合のアニメーションが窮屈になる、速くなりすぎるのを防ぐためです。

Default
Overdrive

Type Breakpoint

コンテナのサイズからRippleのTypeを変更することができます。
クリック時、コンテナのサイズが設定したサイズより小さい場合は通常の"Ripple"を表示し、同じか大きい場合は"Overdrive"を表示するようになる。
