Overview

テーマをセットする方法は、 main.ts 等の初期のタイミングで読み込まれるファイルを使います。
任意のファイルで MlTheming.init([...]) 関数を呼び出します。

以下は、用意されているテンプレートを用いたテーマの設定の一例です。

Wrapper class

theme用のスタイルが、wrapperClassに入力した名前のクラスに囲まれた状態で追加されます。
<body>などのタブに、指定したクラスを付与することでスタイルが反映されます。重複させることはできません。
